# Todo App with Node.js, Express, and EJS

A simple **Todo List Web Application** built using **Node.js**, **Express**, and **EJS**.  
Users can **add**, **view**, **edit**, and **delete** tasks in a clean and interactive interface.

## **Features**

- Add new tasks with unique IDs.  
- View all tasks in a list with automatic serial numbers.  
- Edit existing tasks.  
- Delete tasks.  
- Fully dynamic using **EJS templates**.  
- Uses **Bootstrap** for styling (optional if you add UI).
